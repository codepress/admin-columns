# Admin Columns

Customize columns on WordPress admin screens for posts, pages, media, users, and more. Add, reorder, and remove columns without writing code.

- **Website:** [admincolumns.com](https://www.admincolumns.com)
- **Plugin directory:** [wordpress.org/plugins/codepress-admin-columns](https://wordpress.org/plugins/codepress-admin-columns/)

## Installation

The easiest way to install Admin Columns is from the WordPress admin:

1. Go to **Plugins → Add New**
2. Search for *Admin Columns*
3. Click **Install Now**, then **Activate**

Or download the latest ZIP from the [WordPress plugin directory](https://wordpress.org/plugins/codepress-admin-columns/) and upload it via **Plugins → Add New → Upload Plugin**.
